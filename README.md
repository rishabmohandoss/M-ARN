# Our Pixel Story - Chrome Extension

A cute, interactive Chrome extension featuring pixel art characters with animations. Appears in the corner of every webpage with photo cycling and fun click interactions.

## Features

- 🎮 **Pixel Art Characters** - Adorable pixel versions of us
- 🖼️ **Photo Gallery** - Navigate through 7 favorite moments with arrow buttons
- 🌀 **Girl Animation** - Click to turn circular and roll away with "GRRRR!"
- 🦆 **Guy Animation** - Click to transform into a white duck saying "QUACK!"
- ❤️ **Always Together** - Follows you across the web

## Installation

### Step 1: Download the Files
Download these three files:
- `manifest.json`
- `content.js`
- `content.css`

### Step 2: Create a Folder
Create a new folder on your Desktop named `our-pixel-story` and move all three files into it.

### Step 3: Load in Chrome
1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer Mode** (toggle in top-right corner)
3. Click **Load unpacked**
4. Select the `our-pixel-story` folder
5. Done! The extension will now appear on every website

### Step 4: Use It!
Visit any website and look for our pixel buddies in the bottom-right corner:
- Click **← →** arrows to cycle through photos
- Click the **girl** to see her roll away
- Click the **guy** to watch him become a duck

## Troubleshooting

**Extension not showing?**
- Refresh the extension at `chrome://extensions`
- Close and reopen the tab you want to see it on

**Files won't download?**
- Right-click each file and choose "Save link as"

**Not appearing on some sites?**
- Chrome blocks extensions from internal pages (chrome://, settings)
- Works great on Google, Wikipedia, news sites, etc.

## Customization

Want to modify the extension?
- Edit `content.css` to change colors, sizes, or animation speeds
- Edit `content.js` to add new features or change text messages
- After changes, refresh the extension at `chrome://extensions`
