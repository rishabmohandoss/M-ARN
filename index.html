A cute Chrome extension that brings us to every webpage

What is this?

Our Pixel Story is a custom Chrome extension featuring adorable pixel art versions of us. It appears in the corner of every webpage, with fun interactive animations and the ability to cycle through our favorite photos together.

Always Together

Our pixel buddies follow you across the entire web

1

Download the Extension Files

You'll need three files for this extension. Download them by clicking the buttons below:

📄 manifest.json 📄 content.js 📄 content.css
2

Create a Folder

Create a new folder on your Desktop and name it exactly:

our-pixel-story

Move all three downloaded files into this folder. Make sure all three files are in the same folder.

3

Verify Your Folder

Your folder should contain exactly these three files:

- manifest.json
- content.js
- content.css

⚠️ Do not rename or move files out of the folder

4

Open Chrome Extensions

Open Google Chrome and type this into the address bar:

chrome://extensions

Press Enter. This opens the Extensions page.

5

Enable Developer Mode

In the top-right corner of the Extensions page, you'll see a toggle labeled Developer mode. Turn it ON.

A few new buttons will appear at the top left. This is safe — it just lets you install extensions from your computer.

6

Load the Extension

Click the blue Load unpacked button that appeared at the top left.

A file browser will open. Navigate to and select your our-pixel-story folder (the entire folder, not a file inside it).

Click Select.

✅ You should now see "Our Pixel Story" listed as an installed extension!
7

See It in Action

Go to any website — Google, Wikipedia, your favorite news site, anything. Look at the bottom-right corner of the page.

You should see our pixel art characters with navigation arrows! Try:

  • Click the arrows (← →) to cycle through our photos
  • Click the girl to see her roll away saying GRRRR!
  • Click the guy to watch him turn into a duck saying QUACK!

🔧 Troubleshooting

  1. Not seeing the extension? Go back to chrome://extensions and click the refresh icon (↺) next to "Our Pixel Story". Then close and reopen your current tab.
  2. Files won't download? Right-click each file link and select "Save link as" instead.
  3. It doesn't appear on some websites. That's normal — Chrome blocks extensions from running on its internal pages (chrome://, settings, etc.) and some banking/secure sites. Try Google, Wikipedia, or any regular website.
  4. Animations aren't working? Make sure you're using the latest version of Chrome. Very old versions may have compatibility issues.
